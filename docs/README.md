# `docs/` overview

| File                          | Audience                                         |
| ----------------------------- | ------------------------------------------------ |
| `signal_materiality.md`       | the main pitch — read this first                 |
| `architecture.md`             | contributors extending the codebase              |
| `reproducibility.md`          | anyone running the pipeline end-to-end           |
| `appendix_kittler.md`         | media-studies / philosophy readers (appendix)    |

The top-level `README.md` is the 1-command quickstart. The four files
above are the long form. The paper draft lives in `paper/main.tex`.

## House rules

- Lead with signal materiality, keep Kittler in the appendix.
- Phrase the pipeline as "**resumable semi-auto with 4 explicit
  human-review gates**" — never "fully automatic" or "permanent."
- Do not put R-numbers (e.g. "R14") in commit messages or docs aimed at
  external readers.
